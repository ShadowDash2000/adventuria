Запись значений эффектов в PocketBase представляет собой обычное текстовое поле.
И значение для эффекта обычно записывается в таком формате: `d4;d6;d4` (пример для эффекта замены кубиков).
В PocketBase существуют JSON поля, но такой формат был выбран намерено, так как значения эффектов обычно небольшие и
так просто быстрее и удобнее записывать и читать.

| [add_random_item_to_inventory.go](add_random_item_to_inventory.go) | Добавляет случайный предмет из списка в инвентарь                            | `(string)` item_id1;item_id2;item_id3...            |
